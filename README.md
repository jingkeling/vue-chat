# Vue-Chat

> 项目主要用Vue + VantUI 构建的仿bilbili的webapp。前端项目部署在Docker-nginx容器中




[![forthebadge](https://forthebadge.com/images/badges/made-with-vue.svg)](https://cn.vuejs.org/v2/guide/)



## 技术栈

vue2 + vue-router + vux + websocket
